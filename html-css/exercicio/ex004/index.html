Imagens

Você não pode utilizar qualquer imagem quando for fazer um trabalho profissional, pois as imagens podem possuir licença de uso.

Para pegar uma imagem você vai no Google > Imagens > Ferramentas > Direitos de uso e escolha os filtros

  • Sem filtro de licença (pega independente se tiver dono ou não)
  • Marcadas para reutilização com modificação (pode pegar a imagem e ainda modificá-la para uso comercial)
  • Marcadas para reutilização (pode utilizar comercialmente, mas não pode modificar)
  • Marcadas para reutilização não comercial com modificação (pode usar e modificar, desde que meu objetivo não seja ganhar dinheiro)
  • Marcadas para reutilização não comercial com modificação (pode usar, desde que meu objetivo não seja ganhar dinheiro)
